### chatai
使用vue高仿了chatgpt的前端，后端使用python flask实现。


后端代码
https://github.com/zzy0459/Data_project/tree/master


#### 使用代码（二次开发）
```
# 下载依赖
npm i

# 本地开发
npm run dev

# 打包
npm run build
```

#### 使用代码（直接使用）
dist目录就是打包好的代码，可以直接放在nginx的静态文件下。
